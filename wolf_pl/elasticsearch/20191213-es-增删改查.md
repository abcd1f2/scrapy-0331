- **增删改：**
>       查看分词情况：
>       _analyze
>       {
>           "analyzer":"1k_max_word",
>           "text":"我是你爸爸"
>       }
>       
>       查看所有索引情况：
>           GET _cat/indices?v
>
>       GET查询：
>           GET test1/type1/1  
>               索引/类型/id
>               索引和类型是两个组合成一个唯一的类似索引
>
>       插入数据：
>           PUT test1/type1/1
>           每次put上去的值会覆盖之前的值，如果之前的值，更新的时候没有写，那么更新后字段就没有了，如果之前没有值，则增加值
>
>       更新单个字段的值：
>           post test1/type1/1/_update
>           {
>               "doc": {
>                   "字段名": 更新值,
>                   "字段名2"：更新值2
>               }
>           }
>           post 指令执行修改操作，指定修改的字段内容
>
>
>       删除指定id
>           delete 索引/类型/id
>
>
